HTML INPUT ATTRIBUTES

REQUIRED; The required attribute is a boolean attribute. when present, it specifes that an input field must be filled out before sumbitting the form.

READONLY; The readonly is a boolean attribute which if preasent indicates that the user should not be able to edit the value of the input.

MAXLENGTH; The maxlength attribute specifies the maximum number of characters allowed in the <input> element.

PLACEHOLDER; The placeholder attribute specifies a short hint that describes the expected value of an input field.

STEP; The step input attribute specifies the legal number intervals for an input field usually used for number input type.
